Benötigte terminal installationen um Spiel starten zu können:

        Schritt 1. Windows: pip install --upgrade setuptools; pip install --upgrade --force-reinstall setuptools
        Schritt 1. Mac: pip3 install --upgrade setuptools; pip3 install --upgrade --force-reinstall setuptools


        Schritt 2.: python3 -m venv kls_game_env


        Schritt 3. Windows: kls_game_env\Scripts\activate
        Schritt 3. Mac: source kls_game_env/bin/activate


        Schritt 4. Windows: pip install wheel; pip install socketio
        Schritt 4. Mac: pip3 install socketio


        (Wenn noch nicht geschehen: pip/pip3 install pygame)
